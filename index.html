关于Blender一些实用的插件汇总

发表于 2022-09-06

LazyViewport

我们对物体进行旋转 缩放 位移时, 如果想要让物体沿着某个轴进行变换, 那么需要在键盘中输入对应的X Z Y,

也就是说 我们在对物体进行变换操作的时候, 需要先确定物体当前朝向哪个轴以及哪个轴是我们需要变换的 这一系列思索下来是非常消耗脑力的,反反复复下来很容易晕头转向

通常情况下我们会把更多注意力集中在物体本身, 而不是物体的轴向, 所以为了提升变换效率, 我们可以使用插件 比如lazyviewport

这个插件是国外某位Blender大佬写的, 并将文件放置到了Github上:

Github

当我们将插件安装上之后, 选择物体 输入旋转 缩放 位移三者任意快捷键 则会出现轴向标:

这样一来, 我们就不用关注具体是X Y还是Z轴了, 想往哪个方向拖就往哪个方向

Gift转Mp4 文本转语音 语音生成字幕 视频创作经验分享

发表于 2022-01-14

前言

在公司上班时, 突然心血来潮想制作一个视频, 平常录制视频都是使用OBS Studio, 但是在公司吧, 一来没法录声音 二来没装OBS

此时 我们就得考虑采用一种迂回方案, 那就是图像和声音分开录制, 然后再进行合成

图像的录制我采用的是LICEcap, 这是一个跨平台的屏幕录像工具, 关于这个工具, 可以参考《一款跨平台的Gif录制工具》, 当然你可以直接使用系统自带的录屏工具, 或者QQ录屏也行

声音的话我采用的是文字转语音, 然后顺带将语音生成相应的字幕文件

音视频合成我采用的是blender的视频剪辑功能, 顺带给视频添加了背景音乐

最后的字幕合成我用的是tipard_video_converter_ultimate这款转换工具

以下是制作流程和工具介绍

图像录制

你可以使用以下其中一种工具:

  • OBS Studio : 跨平台录屏直播推流神器, 免费软件, 非常强大, 推荐使用
  • LICEcap: 跨平台gif录制工具
  • 系统自带录屏 : Windows10平台可以使用Xbox Game Bar, Mac平台可以使用截屏工具或者QuickTime
  • QQ录屏: 录屏后可以导出为gif, mp4,mov三种格式, 非常方便

Gif转Mp4

  • CovertIo : 非常好用的在线文件格式转换工具, 可以对图片和视频进行格式转换

文本转语音

  • 讯飞快读 : 1500字以内免费, 利用控台源码查看或者网络抓包可以免注册登录获取音频文件:

  • 音频内录工具: 有些文字转语音工具只提供试听但不提供下载, 如果你确实很满意效果, 可以考虑使用内录工具将音频一边播放一边录制, 或者使用录屏软件也可以, 录完后将音频分离出来

语音转字幕

  • 字幕酱: 一分钟内的语音是免费的, 支持双语

音视频合成

  • Blender: 非常强大的一款3D/2D制作工具, 能建模, 能做动画, 还能视频剪辑和特效, 堪称全能, 最重要的是, 它是免费的

视频添加字幕

你可以使用以下其中一种工具:

  • tipard_video_converter_ultimate 提供Mac下载 （访问密码：312306）
  • HandBrake 支持Windows和Mac

更多关于给视频添加字幕的方式, 可以参考《YouTube视频和字幕下载以及合并的工具推荐》

最后成果展示

总结

虽然不完美, 同时流程也比较繁琐, 但是录制一个应急视频教程是完全没有问题的, 充分利用手头上现有的工具快速进行视频创作

缩短创作周期, 让自己轻松一些, 才能高效产出

做过视频的都知道，BGM是一个视频的灵魂。

1、***FreePD：***

（https://freepd.com）**我自己常用的音乐网站。汇集了互联网海量免版税的公共领域音乐资源，提供了许多不同类型的音乐资源，你可以不花钱就能免费下载任意一首音乐，花个14美元可以一键下载全部内容。

（https://freemusicarchive.org/）**一个强大的免费音乐网站，免注册下载全球开放好音乐。

（https://incompetech.com/music/royalty-free/music.html）\简单直接，免费好用，一键下载，无需注册。**

（https://audiojungle.net/）**这个是Envato Market旗下的一个免费音乐素材网站，*Envato Market*做视频的应该都很熟悉吧，主要是做各种片头模板的，现在淘宝上卖的各种AE模板，一大半来自*Envato Market*。

（https://musopen.org/）**免费的古典音乐大全，可按照作曲家，年代，演奏家搜索。完全免费，没有版权限制（个人和商业），对于这样的网站运营者，只想说一声好人一生平安。

9、Netlabels：\

（https://archive.org/details/netlabels）**超好用的免费音乐合集，完全CC0的免费正版素材，随意使用~

10、 Audionautix：*（https://audionautix.com/）**由视频专业团队精心挑选的免费影视素材视频。良心素材网站！*
