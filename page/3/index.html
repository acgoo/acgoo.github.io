NCM格式在线转MP3网站汇总

发表于 2023-07-12

前言

下载的网易云NCM格式音乐，一般只有使用专属的音乐平台才能播放，剪辑或者用于视频背景音乐都会出现格式问题无法使用，下面给大家分享几个操作简单又免费的音频格式转换工具，就连其他平台的音乐格式也能转换。

在线工具

NCM直接转MP3

http://convert.freelrc.com/NCM-to-MP3

一个格式丰富的在线音频转换网站，网站将所有的相互转换的音频格式进行了整理分类，支持转换的音频格式多达十几种，包括： NCM转MP3、NCM转FLAC、QMC转MP3、QMCOGG转OGG、TKM转M4A、BKCMP3转MP3等。

转换的方法超简单，页面下方的第一个格式转换就是NCM转MP3，点击对应的转换格式后，上传下载的NCM音频文件，上传完成后它会自动将格式转换成MP3，转换成MP3后支持免费下载或者在线播放试听音频效果。

音乐转换网

https://kqwyygszh.pages.dev/ncm/

一个专门用来转换各大平台音乐的网站，基本上下载的音乐文件格式都能用它转换成MP3，网站仅支持转换输出为MP3格式。

打开页面后会默认为NCM格式转换页面，可以直接点击或拖曳的方式上传音频文件，上传后会直接转换成MP4格式，而且上传音频后还能看到文件的歌手、歌名、同原文件名等详细信息。

转换云

3、转换云
https://www.zhuanhuanyun.cn/?to=.zh-warp

这也是一个免费的音乐格式转换网站，能够转换多个平台下载的音乐文件，它的页面干净简洁，打开就可以看到各种转换格式。

打开页面后点击上传需要转换的音频文件，上传文件后可以看到歌曲的封面、名字、歌手、以及收录的专辑，同样会默认转换成MP3格式，转换完成就可以直接下载。

如何制作一款属于自己的可商用字体

发表于 2022-10-26

前言

作为内容创作者, 素材版权问题是一个不可忽视的问题, 无论是视频, 图片,还是字体

总之 到处都得花钱 坑~

为了避免陷入版权纠纷, 如果有特殊字体需求的话, 那么有必要考虑自己制作一款可商用的字体

以手写字体为例,网上有很多手写字体制作平台, 比如flexifont、 我字酷、萝卜工坊等等, 制作起来确实比较方便, 但是不允许商用

自己的字体无法商用, 这就相当于不是自己的, 弄着玩玩还行, 一旦投入创作中, 有可能会出大问题

所以 我推荐使用单独的软件进行制作, 比如PS, FontCreator, FontLab,Glyphs 3, FontForge等等

这里我推荐大家使用FontForge, 因为它免费开源, 这就相当于在保险之外再加了一层保险

FontForge

优点:

  • 免费开源
  • 跨平台 支持win mac和linux
  • 支持中文

点击进入官网

Github

FontForge使用

  1. 配置编码 选择ISO-10646 基础集即可

  2. 选择需要设计的字体语言

    点击元素–>字体信息–>字符集

  3. 开始设计字体

    我们可以直接在窗口中进行字体编辑, 也可以从外部导入已经设计好的图形文件

    如果你采用的是鼠标自由绘制模式, 那么需要进行一个线条加粗设置

    双击手绘曲线工具, 此时会弹出一个设置窗口, 我们勾选上Circular即可:

  4. 导出字体

    设计完成后点击文件–>导出即可 , 这里我们可以选择OpenType格式

视频教程参考: 点击观看视频

中文文档参考: 点击进入

附加

如果你使用的是windows系统, 那么还有一款软件也不错, 名为eduoffice我的字, 适合手写字体制作, 支持数位板和鼠标书写, 内置中文字体模板

不过这款软件优点老了 而且仅支持windows系统, 有兴趣的可以尝试一下

2.5D游戏是怎么个回事

发表于 2022-09-14

前言

我们知道2D游戏和3D游戏, 这个很好区分, 近些年突然出现了一个新词叫2.5D, 这到底指的是什么呢

首先 本质上并不存在2.5D这个东西, 2.5维是不存在的, 只是从视觉角度上我们可以得到一种介于2D和3D之间的效果, 这种视觉效果 我们称之为2.5D

效果

比如以下几种方式搭建起来的效果:

  1. 2D 角色 + 2D 场景，摄像机以斜 45度或则60度视角拍摄 形成一种视觉类3D效果

  2. 2D 角色 + 3D 场景: 摄像机视角固定 形成特别的3D效果 比如HD-2D像素风采用的就是这种:

    截图来源B站

  3. 3D 角色 + 2D 场景 特别的3D效果

  4. 2D 角色 + 2D 场景，但角色可以存在上下移动
