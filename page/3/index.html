Blender视频剪辑之如何实现视频变速效果

发表于 2024-05-04

前言

我们在进行视频剪辑时经常会有对视频进行变速的需求

那么接下来就给大家介绍一下 在Blender中如何实现该效果

实现步骤

  1. 将视频导入blender

  2. 选中视频片段 然后添效果片段–>速度控制

  3. 将绿色的速度片段放在需要调整速度的视频片段上层

  4. 拉伸视频即可调节速度

    速度片段会跟随视频片段同步拉伸 拉长速度变慢 拉短速度变快

  5. 赶紧试试吧😜

其他

除了上面所说的常规手段之外 我们还可以调整输出帧率来实现变速效果 帧率调大视频加速 帧率调小视频降速

两个不同帧率的视频

当我们将两个不同帧率的两个视频进行剪辑时 会出现高帧率视频画面变慢的问题

这种情况很容易让初学者误以为是视频资源占用过高导致画面卡顿

解决这个问题的方法也很简单 我们只需要对其中一个视频进行变速处理直到两个画面相匹配

微信公众号如何修改已经发布的文章

发表于 2024-05-02

前言

对于已发表的文章 微信公众号只支持修改20个字以及最多3张图片的修改

不知道为什么微信公众号会设置这样逆天的规则 我们在进行创作的过程中难免会产生内容增减的需求

既然无法直接修改已经发表的文章 那么我们只能另辟蹊径 通过另外一种方式进行实现

解决方案

我们可以通过重新发表的方式进行修改

是的 你没听错 重新发表 然后删除之前发表的文章

不需要重写一遍文章 也不需要复制粘贴 只需要在草稿箱里 创建新文章 然后将已发表的文章添加进去

这种变相的修改方式虽然很另类 但也只能如此

希望公众号未来能更加脑洞大开😂

Blender视频剪辑之迁移素材目录

发表于 2024-04-26

前言

有时候我们可能会导入不同目录下的视频素材到同一个工程中

当我们要对工程视频素材汇总 或者移动关联的素材时 会出现工程找不到素材的情况

当然了 要避免这种情况的发生也很简单 只需要把所有用到的素材放在先移动到同一个目录后再将它们导入工程

但是我个人的习惯是先剪辑 后整理归档命名

所以 这种迁移素材导致关联失效的问题 需要解决

解决方案

blender已经给我们提供了相应的解决办法

假如已经迁移了素材

只需点击上方菜单栏的文件-->外部数据-->查找缺失文件 即可 如果你用的是英文版 则为File -->External Data --> Find Missing Files

选择素材所在的新目录 Blender会尝试自动定位缺失的素材

假如还未迁移素材

如果准备迁移 但还未进行迁移 那么我们可以使用blender提供的资源打包功能 将所有外部的资源打包当当前工程目录下

不过这种方式好像只能打包视频和音频 图片素材不能打包:

如果你只想打包某一个素材 可以选中素材后进入到右下方属性面板片段-->源 我们可以找到打包按钮:
