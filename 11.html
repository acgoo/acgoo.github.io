Gift转Mp4 文本转语音 语音生成字幕 视频创作经验分享

发表于 2022-01-14

前言

在公司上班时, 突然心血来潮想制作一个视频, 平常录制视频都是使用OBS Studio, 但是在公司吧, 一来没法录声音 二来没装OBS

此时 我们就得考虑采用一种迂回方案, 那就是图像和声音分开录制, 然后再进行合成

图像的录制我采用的是LICEcap, 这是一个跨平台的屏幕录像工具, 关于这个工具, 可以参考《一款跨平台的Gif录制工具》, 当然你可以直接使用系统自带的录屏工具, 或者QQ录屏也行

声音的话我采用的是文字转语音, 然后顺带将语音生成相应的字幕文件

音视频合成我采用的是blender的视频剪辑功能, 顺带给视频添加了背景音乐

最后的字幕合成我用的是tipard_video_converter_ultimate这款转换工具

以下是制作流程和工具介绍

图像录制

你可以使用以下其中一种工具:

  • OBS Studio : 跨平台录屏直播推流神器, 免费软件, 非常强大, 推荐使用
  • LICEcap: 跨平台gif录制工具
  • 系统自带录屏 : Windows10平台可以使用Xbox Game Bar, Mac平台可以使用截屏工具或者QuickTime
  • QQ录屏: 录屏后可以导出为gif, mp4,mov三种格式, 非常方便
  • IShot Pro: Mac平台下好用的截图、贴图、录屏工具
  • GIF Brewery3: Mac平台下的Gif录制工具 支持屏幕录制和摄像头录制 还能将本地mp4视频转成gif格式

Gif转Mp4

  • CovertIo : 非常好用的在线文件格式转换工具, 可以对图片和视频进行格式转换

文本转语音

  • 讯飞快读 : 1500字以内免费, 利用控台源码查看或者网络抓包可以免注册登录获取音频文件:

  • 音频内录工具: 有些文字转语音工具只提供试听但不提供下载, 如果你确实很满意效果, 可以考虑使用内录工具将音频一边播放一边录制, 或者使用录屏软件也可以, 录完后将音频分离出来

  • 微软语言合成助手 : 目前短视频大部分配音解说用的都是这个

  • 记灵配音 : 免费的在线配音工具

  • TTSmaker : 每周30000字符免费额度

语音转字幕

  • 字幕酱: 一分钟内的语音是免费的, 支持双语

音视频合成

  • Blender: 非常强大的一款3D/2D制作工具, 能建模, 能做动画, 还能视频剪辑和特效, 堪称全能, 最重要的是, 它是免费的

视频添加字幕

你可以使用以下其中一种工具:

  • tipard_video_converter_ultimate 提供Mac下载 （访问密码：312306）
  • HandBrake 支持Windows和Mac

更多关于给视频添加字幕的方式, 可以参考《YouTube视频和字幕下载以及合并的工具推荐》

总结

虽然不完美, 同时流程也比较繁琐, 但是录制一个应急视频教程是完全没有问题的, 充分利用手头上现有的工具快速进行视频创作

缩短创作周期, 让自己轻松一些, 才能高效产出
